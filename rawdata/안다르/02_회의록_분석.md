# 안다르 — 회의록 분석

> DEMO 재생용 워크드 예시(가상의 1차 미팅 회의록 기반). 회의록이 있을 때만 생성.

## 1. 최신 흐름 판단
- [추정] 도입 임박 전 **검토 초기** 단계. 반품 운영 외주화 가능성을 탐색 중.

## 2. 고객사 유형 분류
- 자체몰 중심 **D2C 브랜드** + 일부 플랫폼 입점. 핏 민감 애슬레저.

## 3. 명시 니즈
- [사실] "반품 처리 속도를 높이고 싶다."
- [사실] "폐기되는 반품을 줄이고 싶다."

## 4. 숨은 니즈 / 페인
| 페인 | 확실성 | 검증 질문 |
|------|--------|-----------|
| 반품 적체로 시즌 재고가 묶임 | 상 | 평균 재입고 리드타임은? |
| 검수 기준 편차로 CS 분쟁 | 중 | 반품 사유 코드 체계가 있나? |
| 회수 가치 손실(폐기·이월) | 상 | 현재 폐기·리퍼 비중은? (미확보) |
| 운영 인력 부하 | 중 | 반품 전담 인력 규모는? |

## 5. 우려 · 승인 장벽
- [추정] 외주 검수 품질에 대한 신뢰(브랜드 기준 유지).
- [추정] 기존 자체 물류/CS와의 역할 분담.
- [추정] 정산 구조의 예측 가능성.

## 6. 우선순위
1. 반품 적체 → 재고복원 리드타임 단축 (48시간 SLA)
2. 회수 가치 전환 (등급별 재판매)
3. 검수 일관성 (99.99% 정확도)
4. 운영비 예측성 (건당 정산)
